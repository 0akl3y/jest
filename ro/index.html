Ușor de configurat

O soluție de testare completă şi uşor de setat JavaScript. Funcționează la cheie pentru orice proiect React.

Feedback instant

Un mod interactiv și rapid, ascultă modificările fișierelor și rulează doar testele aferente, foarte optim și cu răspuns imediat.

Testarea de imagine

Capturați imaginile arborilor React sau a altor valori serializabile pentru simplificarea testării și pentru analiza modificărilor în timp.

Platformă de testare cu zero configurare

Jest is used by Facebook to test all JavaScript code including React applications. One of Jest's philosophies is to provide an integrated "zero-configuration" experience. We observed that when engineers are provided with ready-to-use tools, they end up writing more tests, which in turn results in more stable and healthy code bases.

Rapid şi izolat

Jest paralelizează rularea testelelor pe mai multe procese pentru a maximiza performanța. Mesajele din consolă sunt tipărite împreună cu rezultatele testelor. Fişierele de testare izolate și starea globală se resetează la fiecare test, așadar nu intervine niciun conflict între teste.

Rapoarte de acoperire a codului de incorporate

Creează cu ușurință rapoarte de acoperire a codului folosind '--coverage'. Nicio configurare sau biblioteci suplimentare nu sunt necesare! Jest poate colecta informaţii de acoperirea codului din întregul proiect, incluzând fișiere netestate.

Zero configurare

Jest este deja configurat când utilizaţi [create-react-app] (https://facebook.github.io/react/blog/2016/07/22/create-apps-with-no-configuration.html) sau [react-native init] (http://facebook.github.io/react-native/docs/getting-started.html) pentru a crea un proiect React sau React Native. Plasați testele într-un director __tests__, sau denumiți fişierele de test cu extensia .spec.js sau .test.js. Oricare ar fi preferința, Jest va găsi şi va rula testele.

Încercaţi!

You can try out a real version of Jest using repl.it. Consider a function, add(), that adds two numbers. We can use a basic test in add-test.js to verify that 1 + 2 equals 3. Hit "run" to try it out!

Librărie puternică pentru mocking

O librărie puternică de mocking pentru funcţii şi module. Creați dubluri pentru componentele React Native folosind jest-react-native.

Funcționează cu TypeScript

Jest funcţionează cu orice limbaj ce se compilează în JavaScript şi se integrează perfect cu Babel şi cu TypeScript prin ts-jest.

Prezentări despre Jest

Echipa şi colaboratorii Jest vorbesc în mod regulat despre Jest şi despre faptul că testarea în JavaScript poate fi încântătoare. Puteți urmări prezentările noastre despre [Building High-Quality JavaScript Tools] (https://developers.facebook.com/videos/f8-2017/building-high-quality-javascript-tools/) la f8 2017 sau despre [Snapshot Testing] (https://www.youtube.com/watch?time_continue=416&v=HAuXJVI_bUs) la React Conf 2017.

În acest [video] (https://egghead.io/lessons/javascript-test-javascript-with-jest) de [Kent C. Dodds] (https://twitter.com/kentcdodds) veţi învăţa cum să instalaţi Jest şi să scrieți primul test unitar.

Cine utilizează Jest?

Jest este utilizat de către echipe de toate dimensiunile să testeze aplicații web, servicii node.js, aplicații mobile şi API-uri.
