Tutorial – Webpack Integration

Webpack integration may be achieved by installing the third-party jest-webpack-alias npm module. The preprocessor example below assumes that you're using ES6 features which require the babel-jest npm module, so we'll install that as well.

npm install --save-dev babel-jest npm install --save-dev jest-webpack-alias

Next, you'll need to set up a preprocessor.

// jest/preprocessor.js var babelJest = require('babel-jest'); var webpackAlias = require('jest-webpack-alias'); module.exports = { process: function(src, filename) { if (filename.indexOf('node_modules') === -1) { src = babelJest.process(src, filename); src = webpackAlias.process(src, filename); } return src; } };

Finally, you'll need to configure package.json to tell Jest where to find the preprocessor.

{ ... "jest": { ... "scriptPreprocessor": "<rootDir>/jest/preprocessor.js", } }

jest-webpack-alias can also support multiple Webpack profiles and alternate Webpack config files, so refer to its README for further package.json options.
