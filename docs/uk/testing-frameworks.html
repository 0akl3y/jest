Тестування інших фреймворків

Хоча і може здаватися, що Jest — це інструмент для запуску тестів для React, насправді це універсальна платформа для тестування, з можливістю адаптуватись до будь-якої JavaScript бібліотеки чи фреймворку. В цій секції ми хочемо перелічити пости та статті спільноти про інтеграцію Jest з різними популярними JS бібліотеками.
