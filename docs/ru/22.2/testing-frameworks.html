Testing Web Frameworks

Многие ошибочно считают, что Jest заточен только на React приложения, но на самом деле это универсальная платформа для тестирования, с возможностью адаптации к любой библиотеке или фреймворку JavaScript. В этом разделе мы поделимся ссылками на статьи и посты из сообщества о том, как интегрировать Jest с другими популярными JS библиотеками.
