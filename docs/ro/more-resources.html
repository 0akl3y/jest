Până acum ar trebui să aveţi o idee bună despre cum poate Jest să facă testarea aplicațiilor uşoară. Dacă sunteţi interesat mai mult de procesul de învăţare, aici sunt câteva resurse conexe care v-ar putea interesa.

Navigați documentația #

  • Aflaţi despre testarea de Imagine, Funcții Mock, şi nu numai în ghidurile noastre detaliate.
  • Migrați teste existente la Jest urmând ghidul nostru de migrare.
  • Aflaţi cum să configuraţi Jest.
  • Priviți întregul API Reference.
  • Depanați probleme cu Jest.

Invațați prin exemple #

Veţi găsi multe exemple de teste în folderul exemple pe GitHub. Puteţi învăța multe, de asemenea, din testele excelente din proiectele React, Relay şi React Native.

Alăturați-vă comunității #

Pune întrebări şi află răspunsuri de la alți utilizatori Jest ca tine. Reactiflux este un chat Discord unde se întâmplă o mulţime de discuţii legate de Jest. Intrați pe canalul #jest.

Urmăriţi contul Jest Twitter şi blog-ul pentru a afla ce se întâmplă în lumea Jest.
